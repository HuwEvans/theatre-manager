# Changelog

## Version 3.4
Auditions shortcode update.
- Added the `TM_Auditions` shortcode to list upcoming auditions in ascending audition-date order.
- Updated plugin version metadata and public version constant to 3.4.

## Version 3.3
Season Builder v1.4 update.
- Added audition date and rich-text audition details to Shows.
- Added those audition fields to the Season Builder so they can be edited from the same two-tab screen.
- Renamed builder `Actions` columns to `Remove` to avoid confusion with saved fields.
- Updated plugin version metadata and public version constant to 3.3.

## Version 3.2
Season Builder v1.2/v1.3 update.
- Added a Media tab for season images, show image/program media, and cast headshots.
- Added the remaining Season and Show fields to the builder, including current/upcoming season flags and full show details.
- Switching between builder tabs now saves before changing tabs.
- Updated plugin version metadata and public version constant to 3.2.

## Version 3.1
Season Builder v1.1 update.
- Removed rows in Season Builder (Shows/Cast/Awards) are now moved to Trash when saving.
- Updated plugin version metadata and public version constant to 3.1.

## Version 3.0
Season Builder first release.
- Added Season Builder admin page to manage Season, Shows, Cast, and Awards in one screen.
- Added linked save flow to create/update related data from a single form.
- Added Awards CPT and integrated award editing into Season Builder.
- Updated plugin version metadata and public version constant to 3.0.

## Version 2.8
Security and stability release.
- Fixed cast image rendering in `tm_show_cast` and `tm_season_cast` when image meta stores attachment IDs.
- Escaped unescaped title output in `tm_shows`.
- Fixed malformed inline style strings and corrected shadow option behavior in season shortcodes.
- Added CSS color sanitization helper and applied it to shortcode-generated style blocks.
- Sanitized/validated display options tab query parameter.
- Updated plugin version metadata and version constant to 2.8.

## Version 2.0
Major release: consolidated shortcodes, added sample content tooling and documentation, moved tests out of plugin, improved image/pdf preview troubleshooting and program preview behavior.
